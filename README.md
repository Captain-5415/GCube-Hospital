# 🏥 GCube Hospital - Hospital Management Frontend

A modern and responsive Hospital Management Frontend Application built using **React.js** and **Vite**. This project provides an intuitive user interface for patients to explore hospital services, view departments, check symptoms, and book appointments.

---

## 📌 Project Overview

GCube Hospital is a frontend web application developed to simulate a real-world hospital management system. The application focuses on providing a clean user experience, responsive design, and reusable React components.

---

## 🚀 Features

* 🏠 Responsive Home Page
* ℹ️ About Hospital
* 👨‍⚕️ Doctors Page
* 🏥 Medical Departments
  * Cardiology
  * Neurology
  * Pediatrics
  * Orthopedic
* 📅 Appointment Booking Form
* 🩺 Symptom Checker
* 📞 Contact Page
* 🌙 Light/Dark Theme Toggle
* 📱 Mobile Responsive Design
* 🎨 Modern UI using React Icons
* 🔄 Nested Routing with React Router

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/your-username/GCube-Hospital.git
```

Navigate to the project folder

```bash
cd GCube-Hospital
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
